Formation profesionnelle

Bienvenue a notre section de formation proffessionelle!

  • La machine
      >
    1. Processeur : Le processeur est l'unité centrale d'un ordinateur chargée d'exécuter les instructions et d'effectuer des calculs. Il agit comme le "cerveau" de l'ordinateur.
      2. >
    2. Cartes : Dans le contexte des ordinateurs, les cartes font référence à divers composants matériels pouvant être ajoutés à un système informatique pour améliorer sa fonctionnalité. Des exemples incluent les cartes graphiques et les cartes réseau.
    3. Stockage : Le stockage concerne les dispositifs et les méthodes utilisés pour stocker et récupérer des données numériques. Cela inclut les disques durs, les disques SSD (à semi-conducteurs) et d'autres dispositifs de stockage.
  • Systemes d'exploitation
    • DOS : Un système d'exploitation basé sur du texte utilisé à ses débuts.
      • >
    • Windows : Le système d'exploitation graphique de Microsoft largement utilisé sur les ordinateurs personnels.
      • >
    • Linux : Un système d'exploitation open source réputé pour sa stabilité et sa flexibilité.
  • Bureatique
      >
    • Word : Un logiciel de traitement de texte utilisé pour créer et éditer des documents.
      • >
    • Excel : Un logiciel de tableur utilisé pour l'analyse de données, les calculs et la visualisation.
      • >
    • Power Point : Un logiciel de présentation pour créer des diaporamas.
      • >
    • Access : Un logiciel de gestion de base de données pour organiser et gérer des informations.
  • Web
      >
    • HTML : Le langage de balisage standard pour la création de pages web.
      • >
    • CSS : Un langage de feuilles de style utilisé pour décrire l'apparence et la mise en forme d'un document écrit en HTML.
